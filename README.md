   ```bash

   ```
2. Navigate to the project directory:
   ```bash
   cd snake-app
   ```
3. Start a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Or using Node.js (with live-server or similar)
   npx live-server
   ```
4. Open your browser and go to `http://localhost:8000`.
